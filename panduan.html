Aplikasi Utama

Selamat Datang!

Aplikasi ini dirancang sebagai "Dosen Pembimbing Virtual" yang akan membantu Anda menyusun berbagai dokumen akademis secara sistematis dari nol hingga siap diekspor ke Microsoft Word.

Privasi Terjamin: Aplikasi berjalan 100% di browser Anda. Data tulisan Anda aman dan tidak pernah dikirim ke server pihak ketiga mana pun selain ke mesin AI pilihan Anda.

PERSIAPAN AWAL: Setup Mesin AI

Sebelum memulai, Anda perlu memasang "otak" untuk aplikasi ini:

  1. Klik Ikon Kunci di sudut kanan layar utama untuk membuka Pengaturan API.
  2. Pilih provider AI yang ingin Anda gunakan (Google Gemini, Mistral, Groq, dll).
  3. Masukkan API Key yang Anda miliki ke dalam kolom yang tersedia.
  4. Buat PIN Keamanan. PIN ini untuk mengenkripsi API Key Anda agar aman tersimpan di browser.
  5. Klik tombol Simpan Aturan.

PANDUAN 5 LANGKAH PENYUSUNAN

1

Pencarian Jurnal (Literatur)

Langkah ini membantu Anda mencari bahan baku penelitian.

  • Masukkan Kata Kunci topik riset Anda di kolom pencarian.
  • Tekan Enter, aplikasi akan membuatkan tautan instan ke database bereputasi.
  • Unduh file PDF jurnal yang relevan ke laptop Anda.
2

Ekstraksi & Kajian Pustaka

AI akan membaca jurnal PDF Anda dan mengekstrak intisarinya.

  • Upload PDF jurnal yang sudah Anda unduh. Sistem akan menyalin teksnya.
  • Tekan tombol Auto API. AI akan menyusun tabel ringkasan jurnal.
  • Klik Simpan Jurnal agar data masuk ke memori. Ulangi untuk 3-5 jurnal berbeda.
3

Analisis Komparatif (Mencari Gap)

AI akan membandingkan seluruh jurnal yang sudah Anda simpan di Langkah 2.

  • Tekan Generate Analisis Komparatif.
  • AI akan menemukan Research Gap dan Novelty.
4

Rekomendasi Judul

Menentukan arah tulisan Anda.

  • Tekan Generate Judul.
  • AI akan memberikan 5 rekomendasi judul akademis beserta alasan pemilihannya.
  • Pilih salah satu judul hingga kotak menyala kuning sebagai tanda terkunci.
5

Penyusunan Dokumen Bab per Bab

Ruang kerja utama Anda (Instruksi AI di kiri, Editor Visual di kanan).

  • Memulai Penulisan: Klik tombol Auto API pada bab yang ingin ditulis.
  • Auto-Save: Setiap kali Anda berhenti mengetik 1,5 detik di Editor, sistem akan otomatis menyimpan teks Anda.
  • Micro-Editing (Edit Blok): Blok teks tertentu di Editor, lalu gunakan tombol warna ungu (Perpanjang, Perbaiki PUEBI, atau Parafrase).
  • Export RIS: Khusus di Bab Daftar Pustaka, gunakan tombol "Export .RIS" untuk mengunduh daftar referensi ke Mendeley/Zotero.

FITUR KEAMANAN: Cek Originalitas

Di setiap bab Langkah 5, terdapat panel pengecekan plagiasi bawaan:

Mode Cepat (Lokal)

Membandingkan teks Anda secara gratis dengan teks jurnal yang Anda unggah di Langkah 2.

Mode Akurat (Copyleaks)

Scan luas ke database internet (Membutuhkan API Key Copyleaks terpisah).

Tandai & Auto-Perbaiki: Jika terdeteksi plagiat, klik "Tandai di Teks" untuk mewarnai merah kalimat bermasalah. Lalu klik "Auto-Perbaiki" agar AI otomatis memblok dan memparafrase kalimat tersebut.
