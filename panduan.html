Buku Panduan

Panduan lengkap penggunaan Scientific Document Generator (SciDocGen). Asisten riset pribadi Anda untuk menyusun dokumen akademis berkualitas.

Selamat Datang!

Aplikasi ini dirancang sebagai "Dosen Pembimbing Virtual" yang akan membantu Anda menyusun berbagai dokumen akademis (Proposal, Skripsi, Jurnal, Makalah, Robotik, dan SLR) secara sistematis dari nol hingga siap diekspor ke Microsoft Word.

Privasi Terjamin: Aplikasi ini berjalan 100% di browser Anda. Data jurnal dan draf tulisan Anda aman dan tidak pernah dikirim ke server pihak ketiga mana pun selain mesin AI yang Anda pilih.

PERSIAPAN AWAL: Setup Mesin AI

Sebelum memulai, Anda perlu memasang "otak" untuk aplikasi ini:

  1. Klik Ikon Kunci di sudut kanan atas layar utama untuk membuka Pengaturan API.
  2. Pilih provider AI yang ingin Anda gunakan (Google Gemini, Mistral, Groq, atau Github Models).
  3. Masukkan API Key yang Anda miliki ke dalam kolom yang tersedia.
  4. Buat PIN Keamanan. PIN ini sangat penting untuk mengenkripsi API Key Anda agar aman tersimpan di dalam memori browser.
  5. Klik tombol Simpan Aturan.

PANDUAN 5 LANGKAH PENYUSUNAN

1

Pencarian Jurnal (Literatur)

Langkah ini membantu Anda mencari bahan baku penelitian.

  • Masukkan Kata Kunci topik riset Anda di kolom pencarian.
  • Tekan Enter, aplikasi akan membuatkan tautan instan ke database bereputasi (Google Scholar, IEEE, dll).
  • Unduh file PDF jurnal yang relevan ke laptop Anda.
2

Ekstraksi & Kajian Pustaka

AI akan membaca jurnal PDF Anda dan mengekstrak intisarinya.

  • Upload PDF jurnal yang sudah Anda unduh. Sistem akan menyalin teksnya.
  • Tekan tombol Auto API. AI akan menyusun tabel ringkasan (Tujuan, Metode, Hasil, Limitasi).
  • Klik Simpan Jurnal agar data masuk ke memori. Ulangi untuk 3-5 jurnal berbeda.
3

Analisis Komparatif (Mencari Gap)

AI akan membandingkan seluruh jurnal yang sudah Anda simpan di Langkah 2.

  • Tekan Generate Analisis Komparatif.
  • AI akan menemukan Research Gap (celah penelitian) dan Novelty (kebaruan).
  • Hasil ini akan terus diingat oleh AI untuk menyusun bab-bab selanjutnya.
4

Rekomendasi Judul

Menentukan arah tulisan Anda.

  • Tekan Generate Judul.
  • AI akan memberikan 5 rekomendasi judul akademis beserta alasan pemilihannya.
  • Pilih salah satu judul. Kotak judul akan menyala kuning sebagai tanda terkunci.
5

Penyusunan Dokumen Bab per Bab

Ruang kerja utama Anda (Instruksi AI di kiri, Editor Visual di kanan).

  • Memulai Penulisan: Klik tombol Auto API pada bab yang ingin ditulis. AI akan mengetik secara streaming langsung ke dalam Editor.
  • Auto-Save: Setiap kali Anda berhenti mengetik selama 1,5 detik di Editor, sistem akan otomatis menyimpannya (Tombol hijau "Tersimpan Auto").
  • Micro-Editing (Edit Blok): Blok teks tertentu di Editor, lalu gunakan tombol Magic ungu (Perpanjang, Perbaiki PUEBI, atau Parafrase).
  • Export RIS: Khusus di Bab Daftar Pustaka, gunakan tombol "Export .RIS" untuk mengunduh daftar referensi ke Mendeley/Zotero.
  • Download: Di bab paling bawah, atur Margin/Font lalu unduh menjadi file Microsoft Word (DOCX).

FITUR KEAMANAN: Cek Originalitas

Di setiap bab Langkah 5, terdapat panel pengecekan plagiasi bawaan:

Mode Cepat (Lokal)

Membandingkan teks Anda secara gratis dan cepat dengan teks jurnal yang Anda unggah di Langkah 2.

Mode Akurat (Copyleaks)

Melakukan scan luas ke database internet (Membutuhkan pengaturan API Key Copyleaks terpisah).

Tandai & Auto-Perbaiki: Jika terdeteksi plagiat, klik "Tandai di Teks" untuk mewarnai merah kalimat yang bermasalah. Lalu klik "Auto-Perbaiki" agar AI otomatis memblok dan memparafrase kalimat tersebut hingga aman (hijau).

Tips & Fitur Ekstra

  • Edit Prompt (Kustomisasi): Merasa instruksi bawaan AI kurang pas? Klik tombol Edit di kotak kiri atas pada Langkah 5 untuk mengubah aturan prompt secara permanen untuk dokumen tersebut.
  • Mode Gelap (Dark Mode): Bekerja di malam hari? Klik Ikon Bulan di pojok kanan atas aplikasi untuk mengubah tampilan menjadi gelap.
  • Sistem Navigasi Dokumen: Untuk mengubah mode dokumen (dari Proposal menjadi Skripsi, Makalah, atau Robotik), klik tab mode di menu navigasi Langkah 5. AI akan menyesuaikan struktur bab dan instruksinya seketika.
  • Backup & Restore: Untuk memindahkan pekerjaan ke laptop lain, klik tombol Download Backup (Ikon Disket) di ujung kanan. Pindahkan file .json tersebut, lalu klik Restore Data (Ikon Panah Berputar) di laptop baru.
