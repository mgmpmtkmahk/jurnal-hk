Cara Menggunakan Scientific Document Generator

Selamat Datang!

Aplikasi Scientific Document Generator dirancang sebagai "Dosen Pembimbing Virtual" yang akan menuntun Anda menyusun berbagai dokumen akademis (seperti Proposal, Skripsi, Jurnal, hingga Makalah) secara terstruktur. Berbeda dengan sekadar mengobrol bebas dengan ChatGPT, aplikasi ini menggunakan metode Reverse-Engineering: Anda mengumpulkan bahan baku jurnal aslinya terlebih dahulu, lalu AI akan menganalisis celah penelitian (Research Gap), merekomendasikan judul yang ilmiah, dan memandu Anda menulis bab demi bab yang bebas dari halusinasi AI.

Privasi Data 100% Terjamin

Aplikasi ini berjalan sepenuhnya di dalam browser (komputer) Anda. Teks, jurnal, maupun ide penelitian Anda disimpan di memori lokal browser (Local Storage) dan tidak pernah dikirim ke server kami. Data Anda hanya dikomunikasikan secara langsung ke server penyedia AI (seperti Google/Mistral) menggunakan kunci API rahasia milik Anda sendiri.

1. PERSIAPAN AWAL: Memasang "Otak" AI

Aplikasi ini hanyalah sebuah kerangka kerja. Agar ia bisa berpikir, Anda perlu memberinya "otak" berupa API Key. Ikuti langkah ini sebelum memulai:

  1. Klik Ikon Kunci di menu samping kanan layar utama untuk membuka Pengaturan API.
  2. Pilih Provider AI: Anda bisa memilih Google Gemini (Rekomendasi), Mistral, Groq, atau Github Models.
  3. Masukkan API Key: Dapatkan kode rahasia (API Key) secara gratis melalui tautan yang disediakan di bawah kolom input, lalu tempel (paste) di kolom tersebut.
  4. Atur Temperature (Kepatuhan AI): Pilih angka rendah (misal: 0.3) agar AI patuh pada instruksi pembuatan format akademis yang kaku. Jika Anda memilih angka tinggi (0.7 - 1.0), tulisan akan menjadi puitis namun berisiko merusak format aturan anti-plagiasi.
  5. Buat PIN Keamanan: Ini wajib. PIN ini berfungsi untuk mengunci dan mengenkripsi API Key Anda di dalam browser, sehingga orang lain yang meminjam laptop Anda tidak bisa mencuri API Key tersebut.

2. PANDUAN 5 LANGKAH PENYUSUNAN

1

Pencarian Jurnal (Mencari Bahan Baku)

Tulisan yang baik berawal dari referensi yang kuat. Aplikasi ini menyediakan mesin pencari terintegrasi ke portal jurnal nasional dan internasional.

  • Ketik topik penelitian Anda di kolom kata kunci.
  • Tekan Enter atau klik Cari. Layar akan otomatis bergeser turun menampilkan pilihan portal seperti Garuda Ristekdikti, Neliti, Moraref, atau Google Scholar.
  • Klik portal yang Anda tuju, unduh file PDF jurnalnya, dan kembali ke aplikasi.
2

Ekstraksi Kajian Pustaka

Di sinilah keajaiban AI dimulai. Anda tidak perlu membaca puluhan halaman PDF secara manual.

  • Klik Pilih File PDF untuk mengunggah jurnal yang baru saja Anda unduh. Sistem akan mengekstrak seluruh teks di dalamnya.
  • Klik tombol ungu Analisis via AI (Auto API). AI akan merangkum latar belakang, metode, hasil, dan kelemahan jurnal tersebut dalam hitungan detik.
  • Klik Simpan ke Database. Ulangi proses ini minimal untuk 3 jurnal agar data penelitian Anda kaya.
3

Analisis Komparatif & Mencari Gap

Sistem akan membandingkan seluruh jurnal yang sudah Anda simpan di Langkah 2 untuk menemukan celah penelitian yang belum dikerjakan orang lain.

  • Klik Generate Analisis Komparatif.
  • AI akan menyajikan tabel perbandingan dan merumuskan titik Novelty (kebaruan) yang akan menjadi nilai jual penelitian Anda.
4

Rekomendasi Judul Penelitian

Berdasarkan celah (gap) yang ditemukan di Langkah 3, AI akan merumuskan judul yang paling relevan dan kuat.

  • Klik Generate Rekomendasi Judul. Akan muncul 5 opsi judul lengkap dengan alasan metodologisnya.
  • Pilih salah satu judul dengan mengkliknya hingga kotaknya berubah warna menjadi kuning.
  • Catatan: Setelah judul dikunci, menu jenis dokumen di sidebar kiri (Proposal, Makalah, dll) akan otomatis terkunci untuk menjaga konsistensi alur dokumen Anda.
5

Penyusunan Bab (Ruang Kerja Utama)

Ini adalah fitur inti aplikasi. Setiap bagian dokumen (Latar Belakang, Metode, dll) dipecah ke dalam blok-blok yang bisa digenerasi satu per satu agar hasilnya fokus dan mendetail.

Mode Generasi Teks

  • Tombol Auto API: AI akan langsung mengetik jawaban secara otomatis ke dalam Editor di sebelah kanan.
  • Mode Manual: Anda menyalin instruksi (prompt) di sebelah kiri, lalu mem-paste-nya ke web eksternal (ChatGPT/Claude), lalu hasilnya Anda copas kembali ke Editor.

Fitur Micro-Editing

  • Anda merasa ada paragraf yang kurang pas? Blok teksnya di Editor.
  • Akan muncul tombol ajaib di bawah: Perpanjang (menambah elaborasi), Parafrase (mengubah susunan kalimat), atau Perbaiki PUEBI (membenarkan ejaan).
Auto-Save: Setiap kali Anda berhenti mengetik selama 1,5 detik, atau menekan tombol `Ctrl+S`, aplikasi akan menyimpan teks Anda secara otomatis agar tidak hilang jika browser tertutup.

3. FITUR ANTI-PLAGIASI

Di bagian bawah setiap Editor pada Langkah 5, terdapat Panel Pengecekan Originalitas. Ini penting untuk memastikan bahwa tulisan yang dihasilkan (atau yang Anda copas) aman dari deteksi sistem seperti Turnitin.

Pengecekan Lokal (Gratis & Cepat)

Sistem akan membandingkan teks yang baru ditulis dengan seluruh teks jurnal PDF yang Anda simpan di Langkah 2. Berguna untuk mendeteksi apakah AI menjiplak mentah-mentah referensinya.

Pengecekan Eden AI (Akurat Basis Web)

Melakukan scan plagiasi kelas berat layaknya Turnitin. Syarat: Anda harus memasukkan API Key Eden AI di menu Pengaturan API (Ikon Kunci) terlebih dahulu.

Cek Web Eksternal

Otomatis menyalin seluruh teks Anda dan membuka tab baru menuju penyedia layanan cek plagiasi gratis di internet (SmallSEOTools).

Tindakan Pasca-Scan

Jika skor plagiasi Anda tinggi (berwarna merah), klik tombol "Tandai di Teks" untuk menyorot bagian kalimat yang terdeteksi menjiplak. Setelah itu, klik tombol "Auto-Perbaiki" untuk menyuruh AI memparafrase khusus bagian yang disorot tersebut secara otomatis.
