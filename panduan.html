Dua Mode Penggunaan Aplikasi

Aplikasi ini sangat fleksibel dan dapat digunakan dengan dua metode di setiap langkahnya:

1. Mode Auto API (Otomatis)

Sistem akan mengetik jawaban AI langsung ke dalam aplikasi secara real-time (streaming).

Syarat: Anda wajib memasukkan API Key Google Gemini (Gratis) melalui ikon Kunci () di sidebar kanan. Catatan: Akun gratis dibatasi 15-20 klik per menit.

2. Mode Manual Tab (Copy-Paste)

Anda meng-copy instruksi (Prompt) dari aplikasi, lalu mem-paste-nya secara manual ke web Google Gemini. Hasil dari Gemini kemudian di-copy kembali ke aplikasi.

Kelebihan: 100% Bebas limit/kuota dan tidak memerlukan pengaturan API Key.

Info: Seluruh teks yang di-generate oleh aplikasi ini telah dipatenkan dengan perintah rahasia (Humanizer) agar terhindar dari deteksi plagiasi dan dijamin lolos uji Turnitin < 5%.
1

Cari Referensi Jurnal

Aksi Anda:

Ketik kata kunci penelitian Anda (misal: kualitas tidur santri), lalu tekan tombol Cari Jurnal. Klik pada salah satu rekomendasi portal (misal: SINTA atau Scopus).

Respon Aplikasi:

Aplikasi akan membuka tab browser baru langsung menuju portal jurnal terkait berdasarkan kata kunci Anda. Silakan *download* 3-5 file PDF jurnal dari web tersebut ke laptop Anda.

2

Kaji Jurnal (Ekstraksi PDF)

Aksi Anda (Memasukkan File):

Klik tombol Upload PDF Otomatis, lalu pilih file PDF jurnal yang baru di-download.

Respon Aplikasi:

Kotak teks akan berkedip merah terang dengan tulisan "Membaca file PDF...". Dalam 1-3 detik, notifikasi Pop-up muncul, dan seluruh teks jurnal tersedot masuk ke kotak.

Jika Pakai Auto API:

  1. Klik tombol Auto API.
  2. Respon: Kotak Hasil (kanan) menjadi kebiruan dan AI mengetik kode JSON secara otomatis.
  3. Tunggu hingga Pop-up Hijau muncul.

Jika Pakai Manual Tab:

  1. Klik Manual Tab (membuka Gemini).
  2. Copy-Paste Prompt Hitam + Teks Jurnal ke Gemini.
  3. Copy balasan JSON dari Gemini.
  4. Paste ke Kotak Hasil di aplikasi.

Aksi Terakhir:

Klik tombol Parse & Simpan Otomatis. Aplikasi akan mengubah kode JSON menjadi tabel rapi di bagian bawah layar.

5

Penyusunan Dokumen (Auto-Memory)

Ini adalah fitur paling ajaib. Aplikasi memiliki ingatan otomatis (Auto-Memory) yang saling menyambungkan antar bab. Pastikan Anda mengerjakannya secara berurutan (dari kiri ke kanan).

A. Memilih Jenis Dokumen

Gunakan sidebar (kiri) untuk mengganti tipe form (Proposal, Makalah, Skripsi, dll). Respon Aplikasi: Tombol menu bab akan langsung berubah menyesuaikan format kampus.

Fitur Tongkat Ajaib (Micro-Editing)

Digunakan jika Anda hanya ingin merevisi satu/dua kalimat tanpa mengulang seluruh bab.

  1. Aksi: Blok (Highlight) kalimat di dalam Editor Visual menggunakan mouse Anda.
  2. Aksi: Klik tombol Parafrase atau Perpanjang di bawah editor.
  3. Respon Aplikasi: Teks yang Anda blok akan diganti sementara dengan penanda loading [‚è≥], lalu segera tergantikan dengan ketikan hasil perbaikan AI secara real-time.

D. Menyimpan & Ekspor

Wajib Klik Tombol Simpan setiap selesai 1 bab agar bisa dibaca oleh bab selanjutnya.

Setelah selesai, klik tab paling ujung (Setup & Export). Atur margin kertas, dan klik Download. File Microsoft Word (.docx) akan langsung terunduh.
